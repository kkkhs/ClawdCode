## 本章预览

本章将实现执行管道和权限系统：

- 权限模型设计
- 确认机制
- 敏感文件检测

## 执行管道流程

```
工具调用请求
    ↓
┌─────────────────┐
│ 1. Discovery    │ → 查找工具定义
├─────────────────┤
│ 2. Permission   │ → 权限检查
├─────────────────┤
│ 3. Confirmation │ → 用户确认（如需要）
├─────────────────┤
│ 4. PreHook      │ → 前置钩子
├─────────────────┤
│ 5. Execution    │ → 实际执行
├─────────────────┤
│ 6. PostHook     │ → 后置钩子
├─────────────────┤
│ 7. Formatting   │ → 格式化结果
└─────────────────┘
    ↓
返回执行结果
```
